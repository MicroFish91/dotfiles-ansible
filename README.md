# Install git

| Ubuntu | Fedora | macOS |
| ------ | ------ | ----- |
| `sudo apt install -y git` | `sudo dnf install -y git` | `xcode-select --install` |

# Clone the repository

```sh
git clone
cd dotfiles
```

# Install Ansible

```sh
./setup/install
```

# Run playbook

```sh
./setup/bootstrap
```
